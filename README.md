# Flutter Blogs App 📰

>  This app was originally built by [Rivaan] as part of his YouTube tutorial.  
> I followed the tutorial and implemented it step-by-step for learning purposes, and plan to extend it with my own features.

A simple blogging application built using **Flutter** with modern clean architecture and state management practices.  
This app includes basic authentication and blog functionality such as:

## ✅ Features

-  Sign Up & Sign In using Supabase Auth
-  Create new blog posts
-  Read and view existing blog posts
-  Clean Architecture structure
-  State Management using `Cubit`
-  Dependency Injection using `get_it`
